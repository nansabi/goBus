GoBus is a bus booking platform built with HTML, CSS, JavaScript, and Node.js.

## Features
- Search for bus routes
- Seat selection
- UPI ID payment and receipt
- Login/Signup system

## How to Run
1. Clone the repo
2. Open terminal
3. Run: `npm install`
4. Start server: `node server.js`
